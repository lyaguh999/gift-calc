Сделал специальный калькулятор подарков для игроков STALCRAFT X.
В нём можно быстро посчитать стоимость подарков по количеству, удобно суммировать “мелкие подарки” и сразу видеть общий итог.
Если у вас есть идеи, что ещё добавить или улучшить (новые функции, удобства, предметы для отслеживания, оформление) пишите, буду рад предложениям и постараюсь внедрить лучшие.

Этот проект держится на двух вещах: литре кофе и моем бесконечном энтузиазме. Я сделал это для души, но, как говорится, "спасибо" в карман не положишь (хотя и оно чертовски приятно!).
Если хотите подкинуть мотивации для новых свершений, можете отправить что-нибудь мне на почту, указав ник NewerTEST. Буду благодарен за любую весточку!
